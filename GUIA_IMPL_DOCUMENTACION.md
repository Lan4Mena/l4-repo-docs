# Guía de Implementación del Portal de Documentación

Esta página queda fuera de la navegación principal y funciona como referencia histórica de implementación.

Las fuentes canónicas del estándar l4 repo docs están distribuidas en las páginas publicadas del portal:

*   **[¿Cómo Documentar?](como-documentar.md):** estructura estándar para documentar repositorios l4 repo docs.
*   **[Arquitectura](docs/arquitectura.md):** modelo centralizado y distribuido del portal.
*   **[Despliegue](docs/despliegue.md):** ambientes, validación local, CI/CD, despliegue y recuperación.
*   **[Catálogo de Repositorios](docs/repositorios.md):** repositorios relacionados en cada repo; inventario completo en este portal central.
*   **[Inteligencia Artificial](docs/inteligencia-artificial/):** implementación basada en estándares abiertos para `AGENTS.md` y Agent Skills.

Si alguna instrucción de esta página entra en conflicto con las páginas canónicas, prevalecen las páginas canónicas.
